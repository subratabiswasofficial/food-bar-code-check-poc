# Food Barcode & Label Analyzer

A full-stack web application that analyzes food products using barcode lookup and OCR-based label processing.

## Features
- Barcode-based product data retrieval
- Integration with Open Food Facts API
- Ingredient & allergen detection
- OCR processing using Tesseract.js
- Secure API endpoints with JWT authentication

## Tech Stack
Node.js, Express, MongoDB, Multer, Tesseract.js, Postman

## Learning Outcome
Improved skills in API integration, OCR processing, backend authentication, and real-world data handling.
